# Upcoming Frontend Features

The roadmap for the User Interface and User Experience.

- [ ] **Real-Time Updates**: WebSockets integration to show matches finishing live without refreshing.
- [ ] **Data Visualization**: Interactive charts for ELO history and win rates (Recharts/Visx).
- [ ] **Dark/Light Mode**: User preference toggle with persistent state.
- [ ] **Mobile Optimization**: "Install to Home Screen" (PWA) capabilities for easy office access.
- [ ] **Animations**: Micro-interactions and transition effects (Framer Motion) for premium feel.
- [ ] **Command Palette**: `Ctrl+K` menu for quick navigation and actions.
- [ ] **Theme Customization**: Accent color picker for personalized aesthetics.
